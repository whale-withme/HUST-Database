## 关于实验四要求和代码功能对应  
实验指导书上的要求和系统菜单功能对应关系如下：  
- 1）新生入学信息增加，学生信息修改.----> 1.学生信息  
- 2） 课程信息维护（增加新课程，修改课程信息，删除没有选课的课程信息）.----> 2.课程信息  
- 3） 录入学生成绩，修改学生成绩. ----> 3.选课信息 
- 4） 按系统计学生的平均成绩、最好成绩、最差成绩、优秀率、不及格人数.----> 4.学生成绩统计 
- 5） 按系对学生成绩进行排名，同时显示出学生、课程和成绩信息. ---->5.学生成绩排名 
- 6） 输入学号，显示该学生的基本信息和选课信息.----> 1.学生信息
